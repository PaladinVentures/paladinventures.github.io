Quickstart Installation using ZIP File

Quickstart Installation is full backup of demo site, so it contain joomla installation, extensions, template, and content. This type of installation is very recommended for the new site.

There are 2 method to upload your site:

  1. The most healthy for you, upload the all in one zip file, using ftp client software and extract it on file manager in your hosting panel like cpanel or directadmin. or you can use akeeba kickstart, please watch the manual here
  2. If your hosting panel can not extract the zip file and you don't want to use akeeba kickstart, you should extract it first on your PC and upload the extracted files using the ftp client software.

Then you can create your database name, username, and password, after that do the installation, see the images below and follow the steps  :


  1. Please make sure all Pre-installation Check are marked Yes
    There are some hosting don't set Magic Quotes GPC Off, please read this http://docs.joomla.org/How_to_turn_off_magic_quotes_gpc_for_Joomla_3
    If you use shared hosting and cannot found php.ini file, please create .user.ini file on the root of joomla folder and write this code inside
    magic_quotes_gpc = Off
magic_quotes_runtime = Off
magic_quotes_sybase = Off

  2. This is the place where you set your database.

  3. After database installation is complete, please click on "Next Step" button.

  4. Now it's time to set your site data, Site Parameters and Super User Setting.

  5. You should click "Remove the Insalation Directory" button.

  6. Please click "Visit your site's front-end" button.
  7. Please enjoy.

Grid system

Shaken Up uses a grid of 12 responsive columns, similar to Bootstrap.

Your items can use any number of columns, which you will mark by adding the class col-* to the item, where * is any number from 1 to 12, where 1 means that the item will have a width of 1 column (or 1/12th the width of the section), and 12 means that the item will have a width of 12 columns (or the whole width of the section).

In between the columns, also reminiscent of Bootstrap, the gutter width, or the distance between each column is 30px.

After adding your column classes, you need to add the class clearfix to the wrapper which holds the items, in order to reset the floating behavior.

Header

Contains a parallax background, navigation menu (with your logo) and simple inscription with the words "Scroll down", which takes you to the main section of the theme.

If you want to change your parallax background image, go to /css/style.css, find #header-image, and under background, change the URL of the image to the one you want displayed. Note that parallax scrolling is disabled when the browser width is less than 992px.

The code for the navigation menu is the same as the one in the #navigation-wrapper part of the file (which is the navigation which appears when you scroll to the content part of the page), with the small differences being in the IDs of the elements, and it looks like this:

<div id="header-navigation">
            <a id="header-logo" href="index.html"><img src="images/header-logo.png" alt=""></a>
            <div id="header-menu-wrapper">
                <img src="images/mobile-menu-icon.png" class="mobile-menu-icon" alt="">
                <ul id="header-menu" class="clearfix">
                    <li><a href="#about-wrapper">ABOUT</a></li>
                    <li><a href="#portfolio-wrapper">PORTFOLIO</a></li>
                    <li><a href="#team-wrapper">TEAM</a></li>
                    <li><a href="#services-wrapper">SERVICES</a></li>
                    <li>
                        <a href="#blog-wrapper">BLOG</a>
                        <ul>
                            <li><a href="Blog Masonary article">BLOG MASONRY</a></li>
                            <li><a href="Single article">BLOG SINGLE POST</a></li>
                        </ul>
                    </li>
                    <li><a href="#clients-wrapper">CLIENTS</a></li>
                    <li><a href="#contact-wrapper">CONTACT</a></li>
                </ul>
            </div>
        </div>

The logo in the navigation menu is displayed with the <img> element inside the #header-logo element. The navigation menu itself is inside the #header-menu element. If you want the menu item to scroll to the desired section, simply put the section ID in the href part of the <a> element.

If you want to have a submenu for a particular menu item, simply insert a <ul> element inside that <li> menu item, structured the same way as the whole menu is structured.

Main content

First thing you can notice here is an item with the class separator-100. This is simply an item used to create empty space of different size. Predefined are separator-70 and separator-100 which provide 70px and 100px of space respectively, but you can create your own. Check the /css/style.css and look for .separator-70 on details how.

The rest of the content part of the page is separated into different sections, which will be descriped independently.

About us

Almost every section (apart from clients and testimonials) starts the same way:

<section id="about-wrapper" class="clearfix">
    <h2>About us</h2>

It's wrapped in a <section> tag, with the ID that is named like <name of the section>-wrapper and immediately after that is a <h2> tag which provides the title of the section.

 

Items in the "About us" section look like this:

<div class="about-item col-4" data-scroll-reveal>
                <div class="about-icon li_lab" aria-hidden="true"></div>
                <h4>MODERN DESIGN</h4>
                <p></p>
            </div>

First we have an icon. You can use anything you want here, but we've used Linecon icons, which are included in the theme. For the list of all the icons' class names, go to /fonts/linecons/index.html and find the section "CSS Class Names".

After this we have the item title in a <h4> tag and the item text in a <p> tag.

Note also the data-scroll-reveal attribute in the item. If you add this to an item, the item will fade in when you first hover it when scrolling the page. You can add this to any item on the page.

Our team

In this section you can represent your team members. Each item looks like this:

<div class="team-item col-4" data-scroll-reveal>
                <div class="team-item-image">
                    <img src="images/our-team/photo1.jpg" alt="">
                    <div class="transparent-frame"></div>
                    <div class="team-item-hover">
                        <div class="team-item-icons">
                            <a href="#"><i class="fa fa-behance"></i></a>
                            <a href="#"><i class="fa fa-linkedin"></i></a>
                            <a href="#"><i class="fa fa-dribbble"></i></a>
                        </div>
                    </div>
                </div>
                <h4>ELTON JOHN</h4>
                <div class="item-separator"></div>
                <h5>designer</h5>
                <p></p>
            </div>

You can change the image of the team member, by changing the src attribute in the <img> tag inside the .team-item-image part. Hovering the image gets you the social media for each team member, which you can change in the .team-item-icons part by adding or removing items as styled in the code. We use Font Awesome icons for this part. To get a Font Awesome icon, the <i> tag needs to contain classes "fa" and "fa-*", where you will exchange the * for the name of the icon. For a full list of all icons, visit the Font Awesome list of icons.

After that, the <h4> tag contains the name of the team member, <h5> tag his position/title, and the <p> tag a short description.

Parallax numbers

This section contains a parallax background with numbers that will increase once from zero to the desired number when the section comes into viewport.

You can change the background image in the /css/style.css file, by searching for #parallax-numbers-image and replacing the URL in the background attribute with the URL of your image.

Each of the items here looks like this:

<div class="parallax-numbers-item col-3">
                        <p class="parallax-numbers-number" data-max="325">0</p>
                        <p class="parallax-numbers-text">DESIGNS CREATED</p>
                    </div>

In the .parallax-numbers-number element, the HTML should always be 0, while the data-max attribute should be the value you want your number to achieve after the animation is finished. In the .parallax-numbers-text element, add text which you want to appear under the number.

In the <img>, changing the src attribute, you change which image will appear in the background when you hover the item, and the <h3> tag determines which icon will scroll in from the top at the same time.

Testimonials

Another section with a parallax background. You can change the background image by finding #testimonials-image in the /css/style.css file.

Testimonials look like this:

<div class="testimonials-item">
                        <p>"I've seen things you people wouldn't believe. Attack ships on fire off the shoulder 
of Orion. </p>
                        <h4>Blade Runner</h4>
                    </div>

Put the text in the <p> tag, and the author in the <h4> tag. You don't have to worry about the dots, they will be created automatically.

Clients

In this section you can add your clients' logos.

Each item looks like this:

<div class="clients-item col-2">
                <img class="clients-background" src="images/clients/photo1.jpg" alt="">
                <div class="transparent-frame transparent-80"></div>
                <img class="clients-logo" src="images/clients/envato.png" alt="">
            </div>

The .clients-background item contains the image which is displayed in the background. .clients-logo item containts the client's logo, which is displayed over the background image. Change src attributes of these elements, to change the image.

Footer

In the footer part of the page, you can add your company's logo, trademark text and your social media. It looks like this:

<div id="footer-wrapper">
        <div id="footer-logo-wrapper">
            <img id="footer-logo" src="images/header-logo.png" alt="">
        </div>
        <p>© Shaken Up - a Joomla one page theme. Created by Joomlastars.</p>
        <div id="footer-social" class="clearfix">
            <a href="#"><i class="fa fa-flickr"></i></a>
            <a href="#"><i class="fa fa-facebook"></i></a>
            <a href="#"><i class="fa fa-behance"></i></a>
            <a href="#"><i class="fa fa-twitter"></i></a>
            <a href="#"><i class="fa fa-dribbble"></i></a>
        </div>
    </div>

The <img> tag holds your logo. In the <p> tag you can add your trademark or some short text. The .footer-social part contains the social media icons. Change the Font Awesome class and the href attribute of the <a> tag to suit them to your needs.
